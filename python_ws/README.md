# Python Workspace

This directory contains the ROS 2 package template for ament_python.

## Build/Run Instructions

1. To build the package, navigate to the `python_ws` directory.
2. Run the following command:
   ```bash
   colcon build
   ```
3. Source the setup file:
   ```bash
   source install/setup.bash
   ```
4. Run the node:
   ```bash
   ros2 run template_py_pkg template_node
   ```
